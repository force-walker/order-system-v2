# MVP E2E Checklist (Frontend)

## Environment
- [ ] `./start-dev.sh` で backend/frontend 起動
- [ ] `http://127.0.0.1:8000/health` が 200
- [ ] フロントHeaderの branch 表示が作業ブランチ名

## Core flow
- [ ] 注文作成: ヘッダー + 明細入力で保存できる
- [ ] 注文一覧: 作成した注文が表示される
- [ ] 注文編集: ヘッダー変更、明細追加/更新/削除して保存できる
- [ ] 再表示: 保存内容が反映される

## Master flow
- [ ] 顧客作成: 作成後一覧へ表示
- [ ] 顧客編集: 名前・有効フラグ更新が反映
- [ ] 商品作成: 作成後一覧へ表示
- [ ] 商品編集: 名前/UOM/フラグ更新が反映

## Error/UX
- [ ] 422入力エラー時に画面で意味が分かる
- [ ] 404対象なし時に画面で意味が分かる
- [ ] NetworkError時に接続確認の案内文言が出る
- [ ] 保存成功時に一覧画面でトースト表示
